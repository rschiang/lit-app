Lit
===

Lit is a Qt-based application aimed to make live text projecting like a breeze.

Targets Qt 5.4 or higher.

Supported syntaxes
------------------
```
*italics*
**bold text**
`code segments`
~~strikethrough~~
```

Functionalities
---------------

`Cmd+L` toggles **Code Mode**, which will render text in monospace.
